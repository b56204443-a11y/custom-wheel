# 今天吃什么 - 幸运转盘

一个基于 Canvas 的幸运转盘决策工具，帮你解决"今天吃什么"的选择困难症。

## 功能

- **幸运转盘**：权重随机抽取，带缓动动画效果
- **自定义选项**：添加、删除、修改选项名称和权重
- **颜色自定义**：每个选项可自定义颜色
- **背景图片**：支持上传自定义背景图，可调透明度和模糊度
- **转动记录**：记录每次转动的日期、时间和结果，支持清空
- **深色模式**：一键切换浅色/深色主题
- **数据持久化**：选项、背景、历史记录保存在 localStorage
- **移动端适配**：响应式布局，支持 PWA 添加主屏幕

## 使用方式

直接用浏览器打开 `转盘.html` 即可运行，无需构建或安装依赖。

## 技术栈

- HTML5 Canvas 2D
- 原生 JavaScript（无框架）
- CSS 变量主题切换
- localStorage 数据持久化
